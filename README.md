# 📦 Controle de Estoque em Python

Este é um sistema desenvolvido em Python focado no gerenciamento, monitoramento e controle de fluxo de itens armazenados. O projeto foi estruturado com o objetivo de aplicar boas práticas de lógica de programação, estruturas de dados e manipulação de arquivos ou fluxos no terminal, ideal para otimização de registros e auditorias internas.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **Linguagem:** Python 3.x
* **Ambiente de Desenvolvimento (IDE):** PyCharm
* **Controle de Versão:** Git & GitHub

---

## 🚀 Funcionalidades Atuais

* **Cadastro de Itens:** Registro de novos materiais com propriedades específicas (como nome, quantidade, patrimônio ou número de série).
* **Atualização de Estoque:** Entrada e saída de produtos com validação de quantidades.
* **Consulta e Relatórios:** Listagem detalhada das condições dos equipamentos em estoque.
* **Filtros Automatizados:** Busca rápida por status ou desvios de inventário.

---

## 📂 Estrutura do Repositório

O repositório é composto pelos seguintes arquivos principais:

* `script.py`: Script principal contendo o núcleo da aplicação e gerenciamento do fluxo do sistema.
* `script1.py`: Módulo auxiliar ou de testes para validação das funções e lógicas complementares.
* `.gitignore`: Arquivo de configuração para ignorar pastas locais do ambiente de desenvolvimento (como `.venv` ou `.idea`).
